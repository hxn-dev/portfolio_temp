Nutrinax

Une application mobile cross platform développé avec le framework "flutter" et le langage "dart" !
Elle permet d'avoir un récapitulatif de ses repas et des calories consommées, il suffit de scanner le code bar de son repas (ou des ingrédients) et de renseigner la portion que vous allez consommer.
Une fois scanné l'application se charge d'appeller une API externe pour ainsi recevoir les valeurs nutritives (kcal, glucide, protéine, sel, ...)

BallMyAss

Un jeu dans le genre de Flappy Bird développé avec le moteur de jeu Unity, vous incarnez une balle et vous devez monter le plus haut possible sans toucher les plateformes.
Les plateformes sont réutilisés(object pooling), placé aléatoirement au fur et a mesure que le joueurs monte dans le niveau, ce qui évite le processus lourd de création et suppression des plateformes.
Les touches Flèche gauche et Flèche droite du clavier vous permettrons de bouger de gauche à droite et la barre d'espace / clique droit de la souris pour sauter (Xbox 360 Controller: Stick analogique gauche, bouton A et Start pour recommencer, PS4 Controller: Stick analogique gauche, bouton Carré et R2 pour recommencer)
La difficultée à été fortement diminué pour cette version téléchargeable ;)
