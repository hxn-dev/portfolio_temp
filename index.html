Nutrinax

Une application mobile cross platform développé avec le framework "flutter" et le langage "dart" !
Elle permet d'avoir un récapitulatif de ses repas et des calories consommées, il suffit de scanner le codebar de son repas (ou des ingrédients) et de renseigner la portion que vous comptez consommer.
Une fois scanné l'application se charge d'appeller une API externe pour ainsi recevoir les valeurs nutritives (kcal, glucide, protéine, sel, ...)
